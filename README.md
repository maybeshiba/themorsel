# 🍽️ Morsel

**A podcast for your meal** — daily deep dives curated by AI, playable in-app. No Spotify, no account, free forever.

Covers: politics & power · economics · science & health · culture & identity  
Style: debate-driven, Vox-style deep dives

---

## Deploy to GitHub Pages (free hosting, ~5 minutes)

### Step 1 — Create a GitHub account
If you don't have one, sign up free at [github.com](https://github.com).

### Step 2 — Create a new repository
1. Click the **+** icon (top right) → **New repository**
2. Name it exactly: `morsel`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload the files
1. On your new repo page, click **uploading an existing file**
2. Upload everything in this folder:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - The `icons/` folder (all 3 files inside)
3. Click **Commit changes**

### Step 4 — Enable GitHub Pages
1. Go to your repo **Settings** → **Pages** (left sidebar)
2. Under **Source**, select **Deploy from a branch**
3. Set branch to `main`, folder to `/ (root)`
4. Click **Save**

### Step 5 — Open your app
After ~1 minute, your app will be live at:
```
https://YOUR_GITHUB_USERNAME.github.io/morsel/
```

Open that URL on your Android phone in Chrome and tap **"Add to Home Screen"** when prompted — it installs like a real app!

## Features
- 🎙 Streams real podcast episodes directly in the browser
- 🤖 AI picks the 4 best episodes daily from your favorite shows
- ⏮ Skip back/forward 30 seconds
- 📶 Works offline (app shell cached, needs connection for new episodes)
- 📱 Installable on Android as a PWA (home screen icon, fullscreen)
